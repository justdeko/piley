piley

To-Do app for Android that uses piles to manage your tasks.

Screenshots

Small overview

What you can currently do with piley:

  • Create tasks in a pile and stack them up
  • Create reminders for tasks and get notifications
  • Create new piles to organize your tasks
  • See statistics about piles and tasks
  • Upload backups to your own piley-server instance and use piley on multiple devices
